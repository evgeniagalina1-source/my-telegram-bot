# Telegram-бот на базе GigaChat

Этот проект представляет собой Telegram-бота, который использует GigaChat от Сбер для генерации ответов на сообщения пользователей.

## Установка

1. Убедитесь, что у вас установлен Python 3.8 или выше.
2. Клонируйте репозиторий или скачайте проект.
3. Установите зависимости:

```bash
pip install -r requirements.txt
```

## Настройка

1. Переименуйте файл `.env.example` в `.env` (или создайте новый файл с именем `.env`).
2. Заполните переменные в файле `.env`:
   - `GIGACHAT_API_KEY` - ваш API-ключ для доступа к GigaChat.
   - `GIGACHAT_SCOPE` - область действия (по умолчанию `GIGACHAT_API_PERS`).
   - `TELEGRAM_BOT_TOKEN` - токен вашего Telegram-бота, полученный от [@BotFather](https://t.me/BotFather).

Пример содержимого файла `.env`:

```env
GIGACHAT_API_KEY=ваш_api_ключ_здесь
GIGACHAT_SCOPE=GIGACHAT_API_PERS
TELEGRAM_BOT_TOKEN=1234567890:ABCdefGHIjklMNOpqrsTUVwxyz123456789
```

3. Убедитесь, что у вас есть файл `russian_trusted_root_ca_pem.crt` для настройки SSL-сертификатов (указан в `cert_config.py`).

## Запуск

Для запуска бота выполните следующую команду:

```bash
python run_bot.py
```

Бот начнет принимать и обрабатывать сообщения от пользователей.

## Использование

После запуска бота вы можете отправить ему сообщение в Telegram. Бот передаст ваш запрос в GigaChat и вернет полученный ответ.

## Возможные проблемы

- Если бот не запускается, проверьте, правильно ли указан токен бота в файле `.env`.
- Если запросы к GigaChat не работают, проверьте API-ключ и соединение с интернетом.
